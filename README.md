## 使用
``` javascript
import Vue from 'vue'
import Ymm from '@ymm-design/core'
import '@ymm-design/core/lib/theme-default/index.css'

Vue.use(Ymm)
```

## 开发
`npm run dev` - localhost:8085

## 发布
`npm run pub`

## 文档
`npm run page`

## 测试
`npm run test`

`npm run test:watch`
